Javascript Basic


JavaScript is a cross-plataform, object-oriented scripting language. JavaScript is extermely popular for a variety of reassons. It is a small and lightweight language allowing maximun flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host enviaronment (a web browser or Node server), it can be connected to the objects of these environments to provide programmatic control over them.


  • Variable DeclarationJavaScript variables are containers for storing data values-imagene a cup you fill with coffe, the cup holds the coffee, a variable holds a value. All JavaScript variables must be identified wiyh unique names. These unique names are called indentifiers. var x;
  • Variable AssignmentAssigment operators assing values to JavaScript variables-our cup can now have coffee pourd in it, giving our variable a value to hold. The= assignment operator assigns a value to a variable. x=10;
  • Data TypesData types are an important concept; to be able to operate on variable you need to know the data type. There are six data types that are JavaScript primitives: Boolean-true or false;null-null aka nothing: Number -42 or 3.14159; String- "Coding Dojo Rocks!"; Array- [1, 'Coding',2,'Dojo']; and Object- {fist_name:'Jane',last_name:'Doe'}
  • ConditionalsWhen you write code, you want to perform different actions for different decisions- hitting different code blocks based on values or conditions that have been met. You can use conditional statements in your code to accomplish this. There are the following condition statament: if a specified condition is true, do this code in our code block; else if to specify a new condition to test, if the first condition is false; else we execute this block of code;
  • Loops"There are many different kinds of loops in very programming language, but they all essentially do the same thing: they will repeat an action some number of times. Imagine you have to run a mile, well you run around the track four times and you stop. Thats a loop!
  • FunctionFunctions are an encaspulation of a code block. When we call our function this will run that code block. Thing of it as a list of instructions. As an example imagine we are putting together a desk from Iken, we open up the instruction manual ande get started, first we screw the legs to the table top; next we place the table the right way up. Done! We finished our instructions. Sadly there are a ton more tables to do so lets call our function over and again.
